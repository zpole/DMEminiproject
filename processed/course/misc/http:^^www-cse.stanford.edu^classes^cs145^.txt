CS145

Introduction To Databases

Description

Data models: entity-relationship, relational, network, hierarchical, object-oriented. Relational algebra and calculus, relational database query languages. Dependencies, constraints, and normal forms. Role of databases in application environments. Designing a database for an application. Interactive and programmatic interfaces to database systems. Database transactions from the application perspective. Database application implementation project using a database management system. Prerequisites:
107
,
109B
