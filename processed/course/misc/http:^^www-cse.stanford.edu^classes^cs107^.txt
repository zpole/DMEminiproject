CS107

Programming Paradigms

Description

Introduces a variety of programming language paradigms and their implementations. Topics: structure and implementation of compiled languages, basic concurrent programming, the functional paradigm, and the object-oriented paradigm. Languages include C++ and Lisp. Small programming projects. Prerequisite:
106B
or
106X
.
