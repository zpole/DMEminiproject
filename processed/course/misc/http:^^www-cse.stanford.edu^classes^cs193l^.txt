ProgrammIng in Lisp

Description

Introduction to problem solving in the LISP language, focusing on the functional programming paradigm. Topics: recursion, list manipulation, mapping, functional arguments, destructive processing, macros, I/O, Lisp implementation, environments, packages, efficiency, object-oriented programming, classes, and methods. Term project. Prerequisite:
106B
or
106X
, or equivalent.
