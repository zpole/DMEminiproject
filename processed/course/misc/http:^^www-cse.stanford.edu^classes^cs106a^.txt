CS106A

Programming Methodology

Description

For students in technical disciplines; no prior experience is assumed. Broad introduction to the engineering of computer applications. Software engineering principles are stressed: design, decomposition, information hiding, procedural abstraction, testing, and reusable software components. Uses the programming language C and concentrates on the development of good programming style and on understanding the basic facilities provided by the language. Alternatives:
