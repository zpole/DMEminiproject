CS1U

Introduction To Unix

Description

Tutorial on using the Unix operating system. Topics: the emacs editor, the file system, the C and Bourne shells, and standard Unix utilities (make, awk, sed, grep, etc.). Includes simple shell programming, but it is not a programming course and assumes no prior exposure to programming.
