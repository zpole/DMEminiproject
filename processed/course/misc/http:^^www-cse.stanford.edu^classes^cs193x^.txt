CS193X

X Window System Programming

Description

Develop user interfaces using the X library (Xlib) and the X toolkit intrinsics (Xt). OSF/Motif is used in the examples and programming assignments; emphasis is on general Xlib and Xt techniques. Progressive exercises develop skills in using the X client/server event model, an Xt-based widget set, and more advanced Xlib operators. Prerequisites: experience with C development in a Unix environment.
