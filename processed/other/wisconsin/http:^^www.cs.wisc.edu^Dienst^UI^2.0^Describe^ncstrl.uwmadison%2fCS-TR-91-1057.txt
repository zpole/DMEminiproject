New Programs From Old

New Programs From Old
G. Ramalingam and  Thomas Reps
CS-TR-91-1057
December 1991
There is often a need in the program-development process to generate a new version of a program that relates to existing versions of the program in some specific way. Program- development tools that assist in the generation of the new version, or possibly even automatically generate the new version from the existing versions, will obviously be of great use in this process. The problems of merging software extensions, program integration, separating consecutive edits, and propagating changes through multiple versions are all instances of situations where such tools would be wel- come.
